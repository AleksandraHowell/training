# RDBMS
* Relational Database Management System
* One category of databases
* a relational database is a collection of tables
  * `relation == table`
* Common relational databases
  * PostgreSQL
  * OracleDB
  * MySQL
  * MariaDB
  * SQL Server
* Another category is **NoSQL** (graph, document, key/value)
